How to Use This Slug Generator

Type or paste any text into the input field. The slug updates in real time. Choose between hyphens or underscores as separators, toggle lowercase, and set a maximum length. Accented characters like é, ü, and ñ are transliterated to their ASCII equivalents. All processing runs in your browser — nothing is sent to any server.
